# المرشد القرآني الذكي

المرشد القرآني الذكي هو تطبيق ويب تفاعلي يساعد المستخدم على البحث في القرآن الكريم، الاستماع للآيات، حفظ الآيات المفضلة، متابعة الإنجاز، التسميع بالصوت، واستخدام بوت ذكي للأسئلة المتعلقة بالآيات والتفسير.

## فكرة المشروع

يهدف المشروع إلى توفير تجربة تعليمية وروحية منظمة تساعد المستخدم على قراءة القرآن الكريم والتفاعل معه بطريقة سهلة وحديثة.

التطبيق يجمع بين البحث الذكي، الاستماع، التكرار، التسميع بالمايك، تتبع الأخطاء، والمفضلة داخل واجهة واحدة بسيطة وسهلة الاستخدام.

## المميزات

* البحث الذكي داخل القرآن الكريم
* البحث الصوتي باستخدام المايك
* عرض السور والآيات كاملة
* الاستماع للآيات بأكثر من مقرئ
* حفظ الآيات في المفضلة
* آية اليوم بشكل عشوائي
* وضع التركيز أثناء القراءة
* التحكم بحجم الخط
* وضع استمع وكرر للحفظ
* التسميع بالمايك مع تحليل الأخطاء
* حساب نسبة دقة التسميع
* إظهار الكلمات الصحيحة والناقصة والزائدة
* لوحة إنجاز لمتابعة التقدم
* مسبحة إلكترونية
* شات بوت ذكي للأسئلة الدينية والتفسيرية
* حفظ البيانات محليًا باستخدام Local Storage
* تصميم متجاوب مع الهاتف والحاسوب

## التقنيات المستخدمة

* Next.js
* React.js
* JavaScript
* CSS
* Al Quran Cloud API
* Web Speech API
* Git & GitHub


## طريقة استخدام التطبيق

1. افتح الصفحة الرئيسية.
2. استخدم البحث الذكي للبحث عن آية أو كلمة من القرآن.
3. اختر سورة من القائمة لعرض آياتها.
4. اضغط على زر الاستماع لسماع الآية.
5. احفظ الآيات المهمة في المفضلة.
6. استخدم وضع التسميع بالمايك لاختبار حفظك.
7. تابع تقدمك من لوحة الإنجاز.
8. استخدم الشات بوت لطرح أسئلة عن الآيات أو التفسير.

## ملاحظات مهمة

* ميزة المايك تعمل بشكل أفضل على متصفح Google Chrome.
* يحتاج التطبيق إلى اتصال بالإنترنت لجلب الآيات والصوت من API.
* يتم حفظ المفضلة، التسبيح، والأخطاء داخل المتصفح باستخدام Local Storage.
* الشات بوت يحتاج إلى وجود API داخلي باسم `/api/chat`.
